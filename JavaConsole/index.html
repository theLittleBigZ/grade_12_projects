Number Game

The High/Low number game consists of the user chosing a number between 0 and 100. If the user's guess is too high it'll tell the that their guess was "too high" and if the user's guess is too low it'll them their guess was "too low."The user is allowed to keep guessing until they guess the correct number. The game will keep track of how many times it took the user to guess the correct number. Controlling this game is simple, you use the number keys from the keyboard.

Tic Tac Toe

In this game of Tic Tac Toe the user can play against another user. The first to get 3 of the x's or o's in a vertical, horizantal or diagonal has won the game. You can control this game by using the keyboard and typing in coordinates.

Wanderer

Wanderer is a rougelike game. Were you, the player are placed in a maze looking for the exit along the way your journey will be faced with obstacles that you must learn to overcome. To control this game you must use the following keys: "w","a","s", and "d."
